JSON to SQL converter

Category: Programming

JSON to SQL converter, written in C# and dot.net core, creates relational model and create SQL scripts for multiple databases. This is capable to determine the datatype from JSON and create appropriate data model in sql.

It suports the conversion into following SQL Script:

Suported SQL Scripts:

  1. MSSQL
  2. MYSQL
  3. Oracle
  4. SQLite

Implementation:

  • Step 1 -

    Uploaded file to a folder than refer uploaded file:

    Converter("FileUpload", "C:\json2sql\SampleJSON\sales.json", Convertertype.mssql);

  • JSON Data -

    var json = "[ 100, 500, 300, 200, 400 ]"; Converter("RawData", json, Convertertype.mssql);

For more details - https://github.com/mohdnazir/json2sql#json2sql
